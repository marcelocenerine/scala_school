Sobre

Escola Scala começou como uma série de apresentações no Twitter com o intuito de preparar engenheiros experientes para se tornarem programadores Scala produtivos. Scala é uma linguagem relativamente nova, mas que se baseia em muitos conceitos já conhecidos. Sendo assim, essas apresentações assumiam que o público alvo já conhecia tais conceitos e mostraram como usá-los em Scala. Nós então descobrimos que essa foi uma maneira bastante efetiva e rápida de preparar novos engenheiros. Este site contém o material escrito que acompanhava as apresentações. Consideramos que este material por si só já é bastante útil.

Abordagem

Acreditamos que faz mais sentido abordar o ensino de Scala não como se Scala fosse um Java melhorado, mas como sendo uma linguagem totalmente nova. Não é esperado que você tenha alguma experiência em Java. O nosso foco será colocado no interpretador de código e no estilo objeto-funcional da linguagem, assim como no estilo de programar que temos no Twitter. Também colocaremos ênfase nos aspectos de manutenibilidade e clareza de código, assim como no uso do sistema de tipos.

A maioria das lições requer apenas a instalação do Scala REPL (intepretador interativo). O leitor é encorajado a seguir o material mas também a ir mais longe! Utilize as lições apenas como um ponto de partida para explorar a linguagem.
